# 🛒 E-Commerce Sales Analytics

## 📌 Project Overview

This project is an end-to-end E-Commerce Sales Analytics solution built using Python, MySQL, and Power BI. It simulates a real-world e-commerce business by generating realistic sales data, storing it in a relational database, analyzing it using SQL, and visualizing key business insights through an interactive Power BI dashboard.
